# UI/UX 设计规范与流程

## 1. 导航结构 (App Navigation)

采用底部 **3 Tab** 布局，强调“阅读”与“导入”的分离。

### Tab 1: 书架 (Library) - `(tabs)/index`
*   **定位**: 核心首页，展示已导入的 PDF 文件。
*   **布局**: 
    *   顶部: 搜索栏 (Search Bar)。
    *   主体: PDF 文件列表 (Grid/List 切换)，使用 `FlashList` 优化性能。
    *   无悬浮按钮 (FAB)，保持界面纯净。
*   **交互**:
    *   点击: 打开阅读器。
    *   长按: 触发批量管理模式 (删除、重命名)。

### Tab 2: 导入 (Import) - `(tabs)/import`
*   **定位**: 聚合所有文件获取方式的功能页。
*   **布局**: 纵向排列的卡片式设计 (Card Layout)。
*   **功能模块**:
    1.  **本地导入 (Local File)**: 
        *   点击调用系统文件选择器 (`DocumentPicker`)。
        *   支持多选。
    2.  **网络下载 (URL Download)**:
        *   点击弹出模态框 (Modal)，输入 HTTP/HTTPS 链接。
        *   显示下载进度条。
    3.  **局域网服务 (Wi-Fi Transfer)**:
        *   开关按钮：开启/关闭 HTTP 服务。
        *   状态显示：当前 IP 地址、端口、访问网址 (如 `http://192.168.1.5:8080`)。
        *   提示文案："在电脑浏览器输入上方地址，即可上传 PDF 到手机"。

### Tab 3: 我的 (Settings) - `(tabs)/settings`
*   **定位**: 偏好设置与应用信息。
*   **内容**:
    *   阅读设置：屏幕常亮、默认背景色。
    *   存储管理：清除缓存。
    *   关于应用：版本号、隐私政策。

## 2. 阅读器体验 (Reader Experience) - `reader/[id]`

*   **核心模式**: **垂直连续滚动 (Vertical Continuous Scroll)**。
    *   摒弃传统的左右翻页，采用类似网页长图的浏览方式。
    *   **无缝拼接**: 页面之间无明显间隔，内容连贯。
*   **性能优化**:
    *   **虚拟化渲染**: 仅渲染视口可见区域的页面，上下预加载 1-2 页。
    *   **内存管理**: 滚动出视口的页面自动释放位图内存。
*   **交互细节**:
    *   **单击屏幕中央**: 唤起/隐藏 HUD (顶部标题栏 + 底部进度条)。
    *   **双指缩放**: 支持平滑缩放查看细节。
    *   **暗色模式**: 支持 PDF 颜色反转或滤镜（可选后期功能）。

## 3. 局域网 Web 端界面 (Web UI)

运行在用户电脑浏览器上的简单管理界面。

*   **风格**: 极简设计，响应式布局。
*   **功能区**:
    *   **Header**: 显示 "Free PDF Transfer"。
    *   **Upload Zone**: 拖拽上传区域 (Drag & Drop) 或 "选择文件" 按钮。支持批量上传。
    *   **File List**: 表格展示当前手机内的 PDF 文件 (文件名、大小)。
    *   **Action**: 每个文件后提供 "删除" 按钮。
